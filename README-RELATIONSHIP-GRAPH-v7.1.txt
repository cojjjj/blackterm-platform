BLACKTERM v7.1 // Investigation Graph 2.0

New controls:
- CLUSTER view (default): groups nodes by entity type
- TREE view: case-rooted left-to-right relationship depth
- NETWORK view: expanded radial overview
- Relationship Strength slider: hides weak/noisy edges
- Focus Selected: shows only a selected node and its direct relationships
- Show All: restores the complete graph
- Live search highlighting and auto-focus
- Crowded radial rings expand automatically

This release improves visualization only and performs no additional scanning.
