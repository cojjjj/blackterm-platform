# BLACKTERM X v10.1 // Living Graph

## Relationship Graph upgrades

- Relationship Graph now opens in a deterministic force-directed NETWORK view.
- EXPLORE mode remains available for progressive case expansion.
- Hovering a node highlights its immediate neighbors and fades unrelated context.
- Animated signal particles travel across visible correlation paths.
- A dedicated Entity Inspector shows type, label, risk, detail, and connection count.
- Existing focus, search, threshold, layout, export, linked-case, and global-map actions are preserved.

## Run on Windows

Double-click `RUN_BLACKTERM.bat`.
