BLACKTERM v6.2 — AUTONOMOUS INVESTIGATION ENGINE

Launch:
  python -m pip install -e .
  python -m blackterm_recon gui

Use:
  1. Open Mission Control.
  2. Select NEW INVESTIGATION.
  3. Enter an authorized target and confirm authorization.
  4. Select the stages to run.
  5. Watch Recon, OSINT, Threat Intelligence, AI Correlation, and Report progress live.

Results are stored in one case with evidence, timeline entries, notes, scan links,
and an optional HTML report under ~/.blackterm-recon/reports.

Only assess systems you own or have explicit permission to test.
