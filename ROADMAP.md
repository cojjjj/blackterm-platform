# BLACKTERM Roadmap

BLACKTERM is being developed in public as a modular cybersecurity desktop workspace.

## v6.0 — Living Platform Preview

- [x] Mission Control dashboard
- [x] Live intelligence feed
- [x] Animated counters and activity graph
- [x] Threat and exposure context
- [x] AI analyst panel
- [x] Case/activity timeline
- [x] Toast notifications
- [x] Animated dock and startup sequence
- [x] Persistent event history
- [x] HTML and PDF reporting

## v6.1 — Network Intelligence

- [ ] Interactive topology graph
- [ ] Draggable host nodes
- [ ] Animated scan paths
- [ ] Host profile drawer
- [ ] Service and port overlays
- [ ] Exposure heatmap
- [ ] Timeline replay

## v6.2 — Cases and Evidence

- [ ] Case creation and management
- [ ] Evidence attachments
- [ ] Scan-to-case workflow
- [ ] Notes and findings
- [ ] Case timeline
- [ ] Exportable investigation package

## v6.3 — Intelligence Modules

- [ ] OSINT workspace
- [ ] PhishScan integration
- [ ] PCAP analyzer
- [ ] Threat-intelligence enrichment
- [ ] Hash and artifact analysis

## v7.0 — BLACKTERM OS

- [ ] Draggable and resizable application windows
- [ ] Desktop workspace and animated wallpaper
- [ ] Window snapping, minimize, maximize, and restore
- [ ] Workspace persistence
- [ ] Notification center
- [ ] Unified module launcher
- [ ] Shared cases, telemetry, and AI context

## Long-Term Goals

- Local AI analyst with retrieval over saved scans and cases
- Workflow automation
- Plugin marketplace
- Multi-host correlation
- Attack-path visualization
- Cross-platform packaging
