# BLACKTERM X v10.3 — AI Analyst V2

This drop-in update upgrades the **AI Investigation** tab into a structured analyst workspace.

## Ask the analyst

Try questions such as:

- `Why is this risky?`
- `What changed since the previous scan?`
- `What should I do next?`
- `Explain SMB`
- `Generate an executive brief`

## Trust model

BLACKTERM labels saved observations as facts and contextual interpretation as analyst assessment. Exposure alone is never presented as proof of exploitability.
