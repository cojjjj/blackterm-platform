The map reads geolocated OSINT packages already stored in case evidence. Run an
autonomous investigation or save an OSINT collection to a case, then open:

VISUALIZATION -> GLOBAL INTELLIGENCE MAP

Command bar aliases:
- open global
- open world

Notes:
- Locations are approximate public-source intelligence.
- No new location lookup occurs from the map page itself.
- Markers open a dossier and link directly to the related case.
