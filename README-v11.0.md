# BLACKTERM X v11.0 // Mission Control

This release adds a unified workspace layer on top of the v10.8 visual identity.

## New experience

- Global AI Analyst dock available from every page
- Quick AI actions for summary, risks, and defensive next steps
- Notification Center for platform and intelligence events
- Live status strip showing recon, threat feed, AI, modules, and system time
- New command-bar controls for AI and alerts
- New commands: `ai`, `ask <question>`, `analyst`, `notifications`, and `alerts`
- Ctrl+K entries for Global AI Analyst and Notification Center
- Updated BLACKTERM X v11.0 branding
- Matching purple/cyan glass styling and smooth dock behavior

## Notes

The AI remains grounded in saved BLACKTERM scan data. The notification center uses the connected event store when available and provides safe platform-status fallbacks otherwise.
